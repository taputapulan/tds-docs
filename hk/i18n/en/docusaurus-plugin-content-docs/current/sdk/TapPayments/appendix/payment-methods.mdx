# Supported Payment Methods

TapTap Payments supports 173 countries or regions and 42 currencies, and we are continuously integrating more localized payment methods. Currently, we support the following payment methods globally:

| **Payment Method**      | **Supported Countries and Regions** |
| ----------------------- | ----------------------------------- |
| Credit and Debit Cards  | 173 countries or regions            |
| Alipay                  | 173 countries or regions            |
| PayPal                  | 173 countries or regions            |
| GrabPay                 | Philippines, Singapore, Malaysia    |
| DANA                    | Indonesia                           |
